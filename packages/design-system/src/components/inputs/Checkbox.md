# Checkbox API

## Import

```js
import { Checkbox } from "@neufund/design-system";
```

## Usage

```jsx
<Formik {...}>
 <Checkbox name="tos" label="I agree to the Terms & Conditions" />
</Formik>
```

## Props

| Prop Name | Type    | One of | Default Value |
| --------- | ------- | ------ | ------------- |
| label\*   | string  |        |               |
| name\*    | string  |        |               |
| disabled  | boolean |        |               |
